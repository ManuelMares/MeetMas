Welcome to MeetMas

This extension focuses in professors, providing them with useful features to simplify productivity tasks for them to focus in what is important: class.

What is available in this version:

MeetMas:
Create a file to keep track of the participations. Poll participations randomly and using the smart feature.

MixerMas:
When in any other page (except a meeting or a 'chrome://*' page), get a shortcut for a dice, and two randomizers, one for a list of words and another one for a range of numbers.

MessageMas:
A minimalistic tool that allows you to reply to low-complexity messages in the chat.

Start

Pinning the Extension

To begin using MeetMas, pin it to the browser to have easy access to all the features:
Click in the extension button in your browser to reveal all the available extensions and click in the pin icon for MeetMas

The extension has been added to your browser bar, and you can click on it to access the popup with all the features of the extension

Understanding the Participation Files

MeetMas creates, keeps track and provides you with cvs (Excel) files to register the participation in your meetings. They are known as Participation Files.
You are in control of when the files are created, which file you are using (and where), and what is considered a participation.

Participation Files Menu

This menu appears every time you start a new meeting and it is available only while a meeting is open. This menu might look similar to the popup menu, but while the former one has the main purpose of managing the extension's settings (plus other handy functions), this menu focuses in managing the list of Participation Files only.

This menu can be accessed at any other time while in a meeting, through the button "Participation Files" in the top left menu. This button might been hidden by the collapsed menu

Creating New Participation Files

Open the Participation File Menu, and click in the 'new participation file' button at the bottom. The only requirement to create a participation file is to indicate a new name for it. The link where the file is created will be stored as part of the metadata providing you with easy access to it from the popup menu. This link can be updated. Notice that a single meeting can be used to create as many participation files as you require. Also, you can use any participation file in any meeting you open. However, this is not recommended and it is discouraged.

Load Participation Files

In order to avoid confusion, this option is only available in the popup menu.
While the popup menu is open, the middle section will allow you to load the participation file. Be very carefully in how this file is manipulated before uploading it. The participation file has to be a cvs file, and has to be formatted in the original style. If you have modify it and are not sure of how it should look, you can download a new participation file and copy its structure.
It is also important to notice that the name used to store the file is the one indicated inside the file, in the field 'meet name', instead of the file name.

It is recommended that once the file has been uploaded, you downloaded it again to verify that the format is the expected one, or to make the corresponding corrections.
The order of the participants is not guaranteed to be the same as the one from the uploaded file. That should be the only acceptable difference

Edit the Participation File's Metadata

In the Participation File Menu, identify the file you want to edit, then click in the yellow icon to modify it.
A new menu will show up, allowing you to update the name. It will also give you the option of updating the link to the meet you have open right now.
The link cannot be manually updated to any other value or any other menu to guarantee that the links used are always valid and useful from the popup menu.

Delete Participation Files

In the Participation Files Menu, identify the file you want to edit, then click in the red button to modify it.
Be careful, this action cannot be undone, so it is a good idea to download the file first from the popup, and use it as a backup file in case you need it in the future.

Working with Participation Files

Once you have created and selected a participation file to work with, you can use the interface buttons to simplify your work.

Adding Participants

The list of members provided by Google Meets is the only and necessary way of adding new participants to the registry. The registration occurs automatically, as long as this members menu is visible.
Make sure you open the members menu at least once when new participants are added to the menu.

Registering a participation

This action can be done from the members menu or the chat (also from the MeetMas Menu).
Next to each member's name, you will find out a set of two buttons: a green one to increase the participation by one, and a blue one to decrease it by one. Once a participation is added, a notification will appear confirming the action.
You will notice that you cannot add a participation to yourself from the chat, but it is still possible to do it from the members menu.
It is possible to leave participants with negative total participations.

MeetMas Menu

The MeetMas menu is available only when a participation file is selected. Its goal is to take advantage of the participants register system.
To open the menu, click on the MeetMas Menu button on the top left corner of the meet. The menu will display on the right just as all the native menus do.

Once the menu is open, you can see all the members that have ever been part of you meeting, and keep adding participations. Additionally, new features are provided as button on the top part

Update MeetMas Menu

As mentioned before, all new users can only be registered when the members menu is open. You can use the 'Update MeetMas Menu' shortcut to trigger this action. This button will display the members menu momentarelly, then it will bring back the updated MeeMas Menu.

Random participant

A random member of your meet will be polled. The user will be obtained from the historical list provided in the MeeMas Menu; that is, it will include all the members that have ever been in the meeting, not just the ones that are present in the moment.
An active notification will be displayed to indicate you the name of the polled member.

Smart Random Participant

The users will be sorted to identify the members with less participations. From this resultant list, a random user will be polled
When the user gets selected, an active notification will be displayed.

To exclude particular members from the smart polling, you can add an arbitrary large amount of participations to them, in such way that there are members with less participations than them, and they get excluded from the polling.

Message Mas

Message mas is a minimalistic and simple interface to reply messages in the chat.
While it allows to reply a series of replies, this function is recommended to be used with only one level of reply at a time.

Mixer Mas

Mixer Mas is the only tool that is available in all pages (except those starting with 'chrome://').
This feature allows you roll a dice, randomize a list of words, and randomize a range of numbers in the easiest way ever.
