Welcome to MeetMas

This extension focuses in professors, providing them with useful features to simplify productivity tasks for them to focus in what is important: class.

What is available in this version:

MeetMas: Create a file to keep track of the participations. Poll participations randomly and using the smart feature Add-ons: MixerMas: When in any other page (except a meeting or a 'chrome://*' page), get a shortcut for a dice, and two randomizers, one for a list of words and another one for a range of numbers. MessageMas: A minimalistic tool that allows you to reply to low-complexity messages in the chat. While the MessageMas can be used concatenating several replies, it is recommended to limit its use to a single chained reply.

Setting the extension

To being with, pin MeetMas to the browser to have easy access to all the features:
Click in the extension button in your browser to reveal all the available extensions and click in the pin icon for MeetMas

The extension has been added to your browser bar, and you can click on it to access the popup with all the features of the extension

Participation files

MeetMas creates, keeps track and provides you with cvs (Excel) files for the participation registry in your meetings
You are in control of when the files are created, which file you are using, and what is considered a participation.

Creating new participation files

Everytime you open a meet, a menu will popup, allowing you to select, edit and create a new participation file. This menu can also be accessed from the interface button 'select meet'.
The only requirement to create a participation file is to indicate a new name for it. Although the new created file will store the meeting link where it was created as metadata, you can update this link in the future by chosing to edit the file while in the new meet.
Notice that a single meeting can have as many participation files as you require, and you can use a file in any meeting you open.

Working with participation files

When a meet is started and a participation file is selected from the initial menu, new buttons and menus will appear to help you register participations for the active participants in the most efficient way.

Adding participants to the register

Keep track of all the participants that have ever been in your meet is as easy as keep the members menu open.
Everytime a new participant enters the meet, and as long as the members menu is open, the participant will be registered automatically.
This action only has to be done once per participant. Once the new participant is registered, everything is setup to keep working normally!

Register a participation

There are three sets of buttons that allow you to register participations:

  • Participation buttons from the chat
  • Participation buttons from the members menu
  • Participation buttons from the MeetMas menu
Click on the + button to add a participation, and on the - button to subtract it. When any of the buttons is clicked, a temporary notification will confirm th action You can register as many participation as you want, and the final count will be a balance: total participations = positive participations - negative participations.
Negative total participations are allowed.

Edit a participation file

In the initial menu, click on the yellow button in order to change the name of your file

Delete participation file

In the initial menu, click on the red button of the participation file to delete.
Be careful, this action cannot be undone, so it is a good idea to download the file first in case you need to use in the future as a backup file to create a new participation file

MeetMas Menu

MeetMas menu is available when a participation file is selected in order to take advantage of the participants register system.
To open the menu, click on the MeetMas Menu button on the top left corner of the meet. The menu will display on the right just as all the native menus do.
Once the menu is open, you can see all the members that have ever been part of you meeting, and keep track of their participations. Additionally, new features are provided as button on the top part

Update MeetMas Menu

As mentioned before, all new users can only be registered when the members menu is open. You can use the Update list shortcut to trigger this action. When done, the MeetMas Menu will be displayed automatically again

Random participant

A random member of your meet will be polled. The list of users will be the historical list provided in the menu, not the list of active members displayed in the regular members menu.
An active notification will be displayed to indicate you the polled member

Smart Random Participant

The users will be sorted to identify the members with less participations. From this resultant list, a random user will be polled
When the user gets selected, an active notification will be displayed.

Excluding members from the polling

This feature remains to be implemented due to the logic complexity, but here we provide you with a work around for the Smart Random Participant function: For the users you want to exclude, add a relative large amount of participations.

Download participation files

The downloading of participation files can be done anywhere and at any time, but only from the popup menu.
Click in the extension icon to display the popup, then identify the file you want to download, and click in the download button to obtain the file.
